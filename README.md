# @alkid/alkid-js

TypeScript SDK for Alkid.

## Installation

```bash
npm install @alkid/alkid-js
```

## Usage

```typescript
import { AlkidClient } from '@alkid/alkid-js';

const client = new AlkidClient('your-api-key');
console.log(client.hello());
```

## Development

```bash
# Install dependencies
npm install

# Run in dev mode (with watch)
npm run dev

# Build for production
npm run build

# Type check
npm run typecheck
```
